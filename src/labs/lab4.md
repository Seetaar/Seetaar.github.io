# Лабораторная работа №4: Классификация с применением Scikit-Learn

## Цель работы

Исследовать альтернативные алгоритмы классификации (SVM, k-NN) для задачи предсказания кредитного дефолта и попытаться построить модель с меньшей ошибкой (более высоким ROC-AUC), чем базовая модель Random Forest из исходного задания.

## Ход Работы

1. Реализовать и обучить модели: SVM с RBF-ядром и k-ближайших соседей (k-NN)
2. Провести оценку качества моделей на тестовой выборке с использованием метрики ROC-AUC
3. Сравнить результаты с базовой моделью (Random Forest)
4. Проанализировать причины возможного отсутствия улучшения и сформулировать рекомендации
